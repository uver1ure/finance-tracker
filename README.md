# 💰 Finance Tracker

**A modern glassmorphism personal finance tracker with real-time analytics.**
Built entirely with HTML, CSS, and Vanilla JavaScript.

[🚀 Live Demo](https://uver1ure.github.io/finance-tracker)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 💵 Income & Expense Tracking | Add transactions with categories, amounts, and comments |
| 📊 Interactive Charts | Pie chart (expenses by category) + Bar chart (7-day overview) |
| 🔍 Smart Filters | Filter by Today / Week / Month / All time |
| 🔎 Search | Find transactions by name instantly |
| 📥 CSV Export | Download all your data in one click |
| 🌓 Dark / Light Theme | Toggle between themes |
| 📱 Fully Responsive | Works perfectly on mobile & desktop |
| ⚡ PWA Support | Install as a native app on your phone |

---

## 🛠 Tech Stack

- HTML5
- CSS3 (Glassmorphism, CSS Variables, Flexbox, Grid, Animations)
- Vanilla JavaScript (ES6+)
- Chart.js
- LocalStorage

---
